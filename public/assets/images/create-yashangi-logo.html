How to Create PNG Logo:

  1. Right-click on the logo above
  2. Select "Inspect" or "Inspect Element"
  3. Right-click on the <div class="yashangi-logo"> element
  4. Select "Capture node screenshot"
  5. Save the image as yashangi-logo.png
  6. Or use the SVG file: yashangi-logo-pink.svg and convert it online

Alternative: Use online SVG to PNG converter (see CONVERT_SVG_TO_PNG.md)
